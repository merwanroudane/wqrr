Capabilities

Nine Econometric Methodologies

Migrated from R/MATLAB — unifying QuantileOnQuantile, wqc, nonParQuantileCausality and MATLAB toolboxes into one Python framework.

🔬

Quantile-on-Quantile Regression

Examines dependence between quantiles of two variables using the Sim & Zhou (2015) methodology.

Sim & Zhou (2015)
🌊

Wavelet Quantile Regression

MODWT decomposition + quantile regression across Short/Medium/Long frequency bands.

Adebayo & Ozkan (2023)
📐

Multivariate WQR

Extends WQR to multiple regressors — estimate each variable's effect across quantiles and bands.

Adebayo et al. (2025)
🔮

Wavelet QQR with P-values

Kernel-weighted local polynomial QR on wavelet components with pointwise significance.

Adebayo, Özkan et al. (2025)
⚡

Nonparametric Quantile Causality

Balcilar–Jeong–Nishiyama test for Granger causality in mean and variance across quantiles.

Balcilar et al. (2016)
🌊

Wavelet NP Quantile Causality

Extends causality testing to wavelet-decomposed data at different time horizons.

Balcilar et al. (2016)
🔀

Mediation & Moderation

Five-path wavelet quantile mediation and moderation analysis.

Adebayo (2025)
🔗

Wavelet Quantile Correlation

Quantile correlations between wavelet-decomposed series with Monte Carlo CIs.

Roudane (2024)
📉

Quantile Density Estimation

Wavelet-based nonparametric quantile density estimation with thresholding.

Chesneau, Dewan & Doosti

Get Started

Quick Start

Get up and running in under a minute.

python 
import numpy as np
import wqr

# Generate sample data
np.random.seed(42)
n = 300
x = np.random.normal(size=n)
y = 0.5 * x + np.random.normal(scale=0.5, size=n)

# ── Quantile-on-Quantile Regression ──
result = wqr.qq_regression(y, x)
result.summary()

# Publication-quality visualizations
fig, ax = wqr.plot_qq_3d(result, colormap="jet")
fig, ax = wqr.plot_qq_heatmap(result, colormap="jet")
fig, ax = wqr.plot_qq_contour(result)

# Export LaTeX table
latex = result.export_latex()
print(latex)
